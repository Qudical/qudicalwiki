List of custom types

Not every type of node is listed in the toolbox in the modkit. This is because some nodes are only ever meant to be used once, or just so little in general that giving them their own node in the modkit would just count as unnecessary use of space.

These nodes can still be used in the modkit however, through the Custom Type node. It allows you to enter in the name of any node type, and it will execute that type in its place, provided that that type does not have any properties that need to be set. (As the Custom Type node only has one text field, which is for selecting the type.)
